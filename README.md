# Gemini Tab Summarizer (Chrome 拡張機能)

選択した複数のブラウザタブの内容を抽出し、Google Gemini APIを使用して美しく構造化されたマークダウン形式で一瞬で要約するChrome拡張機能です。また、要約された内容に対してチャットで追加質問を行ったり、Google NotebookLMのソースとして一括追加したり、要約履歴を保存・復元する機能も備えています。

Chrome Manifest V3およびSide Panel APIに準拠した、モダンでプレミアムなダークテーマ設計です。

---

## 🌟 主な機能

1. **Chrome サイドパネル対応**: 閲覧中のタブを切り替えながら、画面端で要約やチャットを行える効率的なワークスペースを提供します。
2. **直感的な複数タブ選択**:
   - 開いている全タブをチェックボックスで個別に選択。
   - キーワード入力によるタブのリアルタイムフィルタリング。
   - 「すべて選択」「選択解除」「アクティブなタブのみ」「同一ドメインのみ」のクイック選択。
3. **スマートなテキストクレンジング**: ページ内のヘッダー、フッター、広告、ナビゲーション、スクリプト等を自動的に排除し、純粋な本文テキストのみを抽出してGemini APIに送信します。
4. **Gemini API 連携 (ストリーミング再生)**:
   - 最新の `gemini-2.5-flash` や `gemini-2.5-pro` などのモデルに対応。
   - ストリーミング生成（SSE）により、文字がリアルタイムに流れるように出力されます。
   - 要約スタイル（箇条書き、要約・結論、詳細分析、複数比較、カスタムプロンプト）を選択可能。
   - 出力言語（自動、日本語、英語）およびクリエイティビティ（Temperature）の調整に対応。
5. **ソース連携（Send to NotebookLM）**: 選択したタブのURLをクリップボードへ一括コピー（改行区切り）し、自動でNotebookLMを開きます。ユーザーは追加画面で貼り付けるだけで、一括ソース追加が可能です。
6. **要約チャット機能**: 要約生成後、そのソース内容に基づいて追加の質問（例：「2つ目のタブと3つ目のタブの共通点は？」など）をチャット形式で行えます。
7. **履歴管理（過去データの復元）**: 最大30件の要約履歴を自動でローカル保存。過去の要約テキストだけでなく、**当時のチャット履歴も完全に復元し、続きから会話を再開可能**です。

## 🛠️ インストール方法

本拡張機能はデベロッパーモードを使用して、Chromeに直接インストールします。

1. **プロジェクトを保存する**:
   本プロジェクトのフォルダ一式を任意の場所に配置します。
2. **Chromeの拡張機能ページを開く**:
   Google Chromeのアドレスバーに `chrome://extensions/` と入力して開きます。
3. **デベロッパーモードを有効化**:
   画面右上にある「デベロッパー モード」のスイッチを **ON** にします。
4. **拡張機能の読み込み**:
   - 画面左上の「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
   - ダウンロード先のフォルダを選択し、「選択」または「開く」をクリックします：
     
5. インストールが成功すると、拡張機能一覧に **Gemini Tab Summarizer** が表示されます。

## 🚀 使い方

### 1. サイドパネルの起動
- Chromeのツールバーにある拡張機能アイコン（紫から青のグラデーションの星型アイコン）をクリックします。
- または、Chrome右上の「サイドパネル」ボタンからプルダウンで「Gemini Tab Summarizer」を選択します。

### 2. Gemini APIキーの設定
- 初回起動時、またはヘッダーの「設定（歯車アイコン）」をクリックすると設定画面が開きます。
- **Google AI Studio** 等で取得したご自身の **Gemini API Key** を入力します。
- デフォルトのモデルや出力言語、Temperature（自由度）を設定し、「Save Settings」をクリックして保存します。
- 現状、使えるモデルは、Gemini 2.5 Flash、2.5 Pro、2.0 Flash、1.5 Flash、1.5 Proとなります。
- 現状、対応言語は、日本語、英語、Autoの3種類です。

### 3. タブの選択と要約
- 現在のウィンドウで開かれているHTTP/HTTPSタブが一覧表示されます。
- 要約したいタブのチェックボックスをオンにします（検索ボックスでの絞り込みや、一括選択ボタンも使用可能）。
- 要約の「スタイル」を選択し、「Summarize Selected Tabs」をクリックすると要約が開始されます。

### 4. 抽出テキストに対するチャット・エクスポート
- 生成された要約はマークダウンで描画されます。
- 「Copy」ボタンでクリップボードへコピー、「Export」ボタンで `.md` ファイルとして保存できます。
- 下部のチャット入力欄から、読み込んだタブの内容について自由に対話できます。

### 5. NotebookLM への送信
- タブを選択した状態で「Send to NotebookLM」をクリックします。
- 自動的にURL一覧がコピーされ、NotebookLMのウェブページが開きます。
- NotebookLMの「ソースを追加」＞「ウェブサイト」を開き、コピーしたURL一覧をペースト（Cmd+V / Ctrl+V）して追加してください。

### 6. 履歴の確認・復元
- ヘッダーの「履歴（時計アイコン）」をクリックすると、過去の要約履歴リストが表示されます。
- カードをクリックすると、当時の要約結果、抽出されたタブコンテキスト、およびチャットスレッドがすべて復元され、続きから会話を行えます。
- 個別にゴミ箱アイコンで削除するか、下部のボタンから一括削除も行えます。

## 📂 ファイル構成

- [manifest.json](manifest.json): Manifest V3 準拠のメタデータ、パーミッション（sidePanel, tabs, scripting, storage）の定義
- [background.js](background.js): ツールバーアイコンクリック時のサイドパネル開閉トリガー制御
- [sidepanel.html](sidepanel.html): UIレイアウト設計
- [sidepanel.css](sidepanel.css): HSLベースのカラー設計、ダークモードテーマ、マークダウン装飾、アニメーション定義
- [sidepanel.js](sidepanel.js): アプリケーションのメインロジック、タブ情報の制御、テキスト抽出、履歴永続化、チャット等の動作制御
- [api.js](api.js): Google Gemini APIへのServer-Sent Eventsストリーミングリクエスト送信モジュール
- [extractor.js](extractor.js): タブ内の不要部分を取り除きテキストを抽出するDOMクレンザー
- [marked.min.js](marked.min.js): マークダウンパーサー（jsDelivr CDNからバンドル）
- [icons/](icons/): 拡張機能用アイコン画像アセット（SVGソースおよびPNGコンパイル版）
